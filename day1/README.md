MiniBlockchain

一个简单的 Python 区块链，自动生成包含随机交易的区块，使用工作量证明（PoW）机制。

功能：
创建创世区块，并持续生成包含 1-10 笔随机交易的区块。
使用 SHA-256 哈希和 PoW（难度=4）。
通过 previous_hash 链接区块。
输出内容包括区块详情、哈希、证明和挖矿耗时。
可以调整 difficulty 改变 PoW 难度。
